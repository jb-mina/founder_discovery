# Founder Discovery — One-Pager

## THE VISION

1인/초기 창업자가 "세상에서 풀 나만의 문제와 고객"을 혼자가 아니라 **전담 AI 코치 팀과 함께 찾도록 돕는 개인용 발견 OS**.

아이디어를 추천해 주는 도구가 아니라, **나 자신을 이해하고, 시장의 문제를 탐색하고, 나와 맞는 문제를 골라 실제 검증까지 이어주는 Discovery Copilot**이다.

---

## 00 — 배경

AI와 도구를 활용해서 마음 먹으면 뭐든 '뚝딱' 만들 수 있는 시대다. 하지만 동시에 내가 무엇을 왜 만들고 그것을 지속가능한 사업의 영역으로 이끌어 가는 건 또 다른 문제고 영역이다.

## 01 — PROBLEM

좋은 아이디어를 찾으려 할수록, 창업자는 종종 **문제가 아니라 솔루션에 끌려가는 함정**에 빠진다.

- 매일 머리로는 "문제/고객부터 보라"는 말을 알고 있지만, 실제로는
    - 트위터, 뉴스, 스타트업 사례, 주변 얘기를 보면 아이디어가 끝없이 생기고,
    - 내가 진짜 풀고 싶은 문제와 그냥 멋져 보이는 문제를 구분하기가 어렵다.
- Self map(나 자신에 대한 이해), Problem universe(세상 문제 지도), Founder-problem fit(나와 문제의 궁합), Validation backlog(검증할 가설 목록)를 **한 몸처럼 다루는 시스템**이 없어서,
    - 결국 그냥 "머릿속 메모 + 노션/노트 조각 + 북마크 + 기억"에 의존하게 된다.
- 그 결과,
    - 아이디어는 늘어나지만,
    - 정말 깊게 들어가 볼 "한 문제"는 잡히지 않고,
    - "무엇을 할지"를 고민하는 시간이 반복된다

---

## 02 — CUSTOMER

**문제/고객 발견을 혼자 붙들고 있는 창업·창업 지향 개인**.

- 1인 창업자, PM 출신 솔로 창업 준비자, 프리랜서/컨설턴트, 커리어 전환을 고민하는 지식 노동자
- 공통점:
    - "나는 뭔가 만들고 싶은데, 무엇을 만들지는 아직 흐릿하다."
    - 스스로와 시장에 대한 이해를 동시에 넓혀야 하는 단계.
    - 여러 도구/템플릿/책은 있지만, **자기 상황에 맞게 이걸 함께 엮어줄 코치/시스템**이 없다.

---

## 03 — CORE MECHANISM

Self map · Problem universe · Founder-problem fit · Validation backlog

이 네 가지가 **끊임없이 순환하며 정교해지는 개인용 발견 시스템**

1. Self map — '나'를 인터뷰하는 Self Insight Agent
    - Agent가 질문을 던져, 관심사·강점·혐오·몰입 경험·경력·접근 가능한 네트워크 등을 인터뷰 형식으로 자연스럽게 끌어낸다.
    - Agent는 고객과의 대화 내용, 답변을 정리해 "나는 어떤 문제에 founder fit이 있는가"를 판단하는 베이스 맵을 만들고 이를 지속적으로 업데이트한다.
2. Problem universe — Seed 스타트업 중심의 문제 지도
    - "세상의 모든 문제"가 아니라, **최근 Seed~Series A 국내외 스타트업이 실제로 풀고 있는 문제**를 중심으로 문제 카드를 수집한다
    - 각 카드에는 누가/언제/왜 겪는지, 현재 대체재가 무엇인지, 무엇이 불편한지까지 같이 기록된다.
3. Founder-problem fit — 문제·사례 후보 큐레이션 + 셀프 평가
    - Self map을 기반으로 Agent가 Problem universe에서 Mina님과 잠재적 fit이 높아 보이는 문제/사례를 추천한다.
    - Mina님이 각 문제에 대해 끌림·이해도·접근성·장기 동기 등을 직접 점수화하고, Agent는 이 피드백을 기반으로 우선순위 상위 1~3개 문제를 정리한다.
4. Validation backlog — 사업/서비스 아이디어 초안 + 검증 플랜
    - 우선순위가 높은 문제에 대해 Agent가
        - "내가 풀 수 있는 방식"의 사업/서비스 아이디어 초안,
        - 1~4주 단위의 검증 플랜(인터뷰, 관찰, 수기 PoC)을 제안한다.
    - Mina님은 여기서 실제 사람과의 인터뷰·실험을 실행하고, 배운 점을 다시 Self map·Problem universe·Fit에 반영한다.

이 네 단계는 일회성이 아니라, **매일/매주 반복되는 루프**로 설계된다.

이 모든 과정과 결과는 잘 기록되어 창업가 Mina의 고유한 자산이자 moat으로 활용된다.

---

## 04 — AGENT ROLES

**각 단계에 특화된 전담 에이전트**를 둔다.

1. Self Insight Agent
    - Mina님에게 관련한 의미있는 질문을 지속적으로 던져 자기 이해를 구조화하고, 메타 인지를 강화하며, Self map DB를 지속적으로 업데이트한다.
2. Problem Scout Agent
    - 국내외 Seed 투자 스타트업·시장 기사·커뮤니티 등을 기반으로 Problem universe 후보를 수집하고, 문제·고객·현재 대체재를 요약한다
3. Fit Judge Agent
    - Self map과 Problem universe를 비교해 "fit 있어 보이는 문제/사례"를 추천 리스트로 만들고, Mina님의 셀프 평가를 돕는다.
4. Validation Designer Agent
    - 상위 문제에 대해 "무엇을 어떻게 검증할지" 인터뷰 질문, 실험 방법, 성공/실패 시그널 등 1pager 기반으로 실행을 설계해 Validation backlog를 채운다.
5. Reality Check Agent (옵션)
    - "이건 창업자의 희망사항일 뿐일 수 있다", "솔루션이 먼저인 것 같다" 등 편향을 지적하고, 더 필요한 증거를 요구한다.

---

## 05 — WHY THIS MATTERS

이 시스템은 "아이디어 생성기"가 아니라, **창업자의 시간과 에너지를 가장 가치 있는 문제로 집중시키는 개인 맞춤형 의사결정 보조 장치**다.

- 창업자는 더 이상 무한한 아이디어/문제 목록 속에서 헤매지 않고,
    - 자신과 진짜 맞는 문제,
    - 이미 시장에서 의미 있게 다뤄지고 있는 문제,
    - 2~4주 단위로 실제 검증 가능한 문제

        에 순차적으로 집중할 수 있다.

- AI Agent는 "생각을 대신"하는 것이 아니라,
    - 생각의 구조를 잡고,
    - 자료 수집과 비교를 돕고,
    - 편향을 짚어주며,
    - 다음 검증 스텝을 제안하는 **디스커버리 파트너**가 된다
- 결과적으로, **"무엇을 할지 고민하는 시간"은 줄이고, "이 문제를 정말 풀어도 되겠다"는 납득과 확신을 쌓는 과정**을 돕는 것이 이 서비스의 핵심 가치다.
